# Microservices Project

Microservices Project is a project created to study microservices concepts, focusing on the use of **Spring Boot**.

### Resources used in the project:
* Feign 
* Ribbon
* Hystrix
* OAuth 
* JWT
* Discovery Server Eureka
* API Gateway Zuul
